Para instalar o git basta acessar o link: https://git-scm.com/downloads, selecionar o sistema operacional utilizado. 
As configs basicas são utilizando os comandos:

git config --global user.name "Seu Nome" - para definir seu nome aos projetos que serão commitados

git config --global user.email "example@email.com" - para definir seu email aos projetos que serão commitados

git config --list - mostra as suas config
